안심번호 서비스 사용 안내

  • 안심번호 서비스는 실제 고객님의 휴대전화번호 대신 1회성의 가상 전화번호인 안심번호 (050*-0000-0000)를 부여해 드리는 서비스 입니다.
  • 택배사에는 실제 고객님의 휴대전화번호 대신 안심번호를 제공하게 됩니다.
  • 안심번호 서비스는 아래 업체에 위탁하여 무료로 제공해 드리며, 이용하신 안심번호는 수취확인 후 일정 기간이 지나면 자동으로 해지됩니다.
* 개인정보 처리 위탁
  • - 위탁 업무 내용 : 안심번호 서비스
  • - 처리 위탁을 받는자 : SK텔링크
  • - 위탁항목 : 휴대전화 또는 일반 전화번호
